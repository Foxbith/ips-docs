## 2. Design Principles

| Principle | Description | Example |
|-----------|-------------|---------|
| Simplicity | Minimize cognitive load | One primary action per screen |
| Consistency | Uniform patterns across the app | Same button styles everywhere |
| Accessibility | Inclusive design for all users | WCAG 2.1 AA compliance |
| Feedback | Clear system status at all times | Loading states, success/error messages |
| Efficiency | Reduce steps to complete tasks | Smart defaults, autocomplete |
